Aplikasi yang pernah Saya buat

Website Checkpoint IoT

Website ini di bangun untuk menyelesaikan tugas akhir atau skripsi saya (Bagian hardware di kerjakan oleh partner saya rida zaimus yang saat itu juga sedang mengerjakan tugas akhir /skripsi).Teknologi yang di gunkanan untuk membangun website ini adalah php native,css,html,javascript dan mysql sebagai database. Pada website ini terdapat dashboard admin yang digunakan untuk mengelola data seperti mendaftarkan,menghapus,edit,dan membuat laporan dalam bentuk document seperti microsoft word dan excel. Pada halaman utama ditampilkan sebuah data pendaki yang sudah di daftarkan dan sebuah halaman peta lokasi alat IoT berupa LoRa dan RFID Reader.Peta pada website ini menggunakan peta opensource dari OpenStreetMap.
